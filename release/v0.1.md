# SemVer-Trust Release Attestation Predicate — v0.1

`https://semver-trust.dev/release/v0.1` is the predicate type for
SemVer-Trust **release attestations**: in-toto Statements binding a
trust-encoded tag and commit to its provenance evidence and release
decision. Defined in [specification §8.1](../spec/semver-trust.md). JSON Schema:
[`schemas/release-v0.1.json`](../schemas/release-v0.1.json).

This emitted predicate is historical. It cannot bind the predecessor-chain,
active/candidate-policy, or authenticated version-state facts required by draft
v0.7/v0.6/v0.5/v0.4 and MUST NOT be used to claim v0.7 release conformance. Draft v0.7
does not change existing v0.1 bytes or fixture expectations; successor
attestations use [`release/v0.2`](v0.2.md). Until coordinated implementation
lands, the legacy emission path is dogfood-only and is not suitable for
production release claims.
