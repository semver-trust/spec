# SemVer-Trust Review Attestation Predicate — v0.1

`https://semver-trust.dev/review/v0.1` is the predicate type for
SemVer-Trust **review attestations**: in-toto Statements generated at
merge time that bind merged commit SHAs to their review facts —
reviewer identities and classes (human/agent), approval verdicts, the
pull-request reference, and the merge strategy used. Defined in
[specification §4.3](../spec/semver-trust.md). JSON Schema:
[`schemas/review-v0.1.json`](../schemas/review-v0.1.json).

This emitted predicate is historical. It cannot bind the profile identity,
canonical actor identity, covered revisions, or final-revision approval state
required by draft v0.5 release evaluation and MUST NOT be used to claim v0.5
release conformance. Draft v0.5 does not change existing v0.1 bytes or fixture
expectations; successor attestations use [`review/v0.2`](v0.2.md).
