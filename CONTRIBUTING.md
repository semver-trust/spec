# Contributing

SemVer-Trust is in early draft (spec v0.9). Feedback is very welcome —
please open an issue or discussion.

Pull requests: contribution terms (CLA or DCO) are being finalized and
will be documented here before external PRs are merged. Until then,
substantive proposals are best raised as issues so they can be discussed
and incorporated by the spec editor.

The specification changes through the editor model described in the
design record; decisions are recorded as ADRs.
